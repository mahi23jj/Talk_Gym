# Talk_gym

Talk_gym is an AI-powered mobile app designed to help people communicate more clearly.

The app focuses on two core skills:
- Improving over-explaining habits by helping users make messages clear, concise, and structured.
- Understanding others' conversations through tone analysis, so users can better interpret intent and emotion.

## Why Talk_gym

Many communication problems are not about vocabulary, but about tone, length, and clarity. Talk_gym helps users practice better messaging in real scenarios and build stronger communication confidence.

## Core Capabilities

- AI communication coach:
	Suggests clearer and shorter phrasing for long or over-detailed messages.
- Tone interpretation:
	Analyzes conversation tone (for example: neutral, supportive, frustrated, urgent) to improve understanding.
- Practice-oriented design:
	Encourages learning through repeated message improvement and feedback.

## Getting Started

### Prerequisites

- Flutter SDK installed
- A configured device/emulator (Android Studio, iOS Simulator, or physical device)

### Run Locally

1. Install dependencies:

```bash
flutter pub get
```

2. Run the app:

```bash
flutter run
```

3. (Optional) Run tests:

```bash
flutter test
```
